Multiple Column

Column Gap

This is a demonstration of the column-gap property. The gap between columns is set to 20px.

Adjusting the gap changes the space between columns. This is useful when creating multi-column layouts.

Column Count

This is a demonstration of the column-count property. The text is split into 3 columns. Changing the value of column-count will control how many columns the text will be split intoChanging the value of column-count will control how many columns the text will be split into.

Column Width

This is a demonstration of the column-width property. The columns have a minimum width of 150px.

With column-width, the content will automatically adjust into as many columns as needed, based on the given width.

Column Rule

This is a demonstration of the column-rule property. A rule (or border) is placed between columns.

The rule's color, width, and style can be customized for visual separation between columns.

Column Span

This is a demonstration of the column-span property. The heading below spans across all columns.

This heading spans across all columns.

Column-span can be used to make a single element span across multiple columns.
